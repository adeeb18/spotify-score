# Spotify Score

## How to Run
Follow the installation instructions below first before running.

Navigate to the repository's directory in command-line and enter:
```bash
npm start
```

## External Dependencies

* react-router-dom
* @emotion/react
* @emotion/styled
* @mui/material
* @mui/icons-material
* react-material-ui-carousel

### Installation Instructions

Navigate to the repository's directory in command-line and enter:
```bash
npm install
```
